# Suggested Commands
- `npm run dev` – Run Astro dev server at http://localhost:4321.
- `npm run build` – Create production build into `dist/`.
- `npm run preview` – Preview the built site locally.
- `npm test -- --run` or `npm run test` – Execute Vitest (jsdom) tests without watch.
- `npm run test:watch` – Run Vitest in watch mode.
- `npm run astro <subcommand>` – Direct access to Astro CLI.
- GitHub Pages deploy workflow: `.github/workflows/deploy.yml` triggered on `main` push or manually.
